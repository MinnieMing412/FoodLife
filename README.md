# FoodLife App

An all-in-one food management app for meal planning, cooking documentation, and restaurant discovery.

## Overview

FoodLife helps you organize all your food-related activities in one place:
- 📅 **Workday Meal Planning** - Weekly calendar with recipes and auto-generated grocery lists
- 📸 **Cooking Photo Gallery** - Document and organize your culinary creations
- 🗺️ **Restaurant Discovery** - Wishlist management with map views and visit logging

## Platform Strategy

- **iOS App**: Complete functionality for meal planning, recipes, grocery lists, photos, and restaurant management
- **Web App**: Minimal interface for photo gallery viewing and restaurant map visualization

## Tech Stack

- **Frontend**: React Native (iOS app) + React Native Web (minimal web interface)
- **Backend**: Go/Golang REST API
- **Database**: PostgreSQL
- **File Storage**: Cloud storage for photos
