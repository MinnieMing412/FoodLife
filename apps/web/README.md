# FoodLife Web

Vite React TypeScript shell for the FoodLife web app.

## Setup

Run commands from `apps/web`.

```bash
npm install
npm run test
npm run build
```

## Stack

- Vite React TypeScript starter
- React Router declarative routes for Home, Made, Found, Timeline, Add Memory, Detail View, and Edit Memory
- Tailwind CSS through `@tailwindcss/vite`
- Vitest with Testing Library for the starter shell test
